# Accessibility

## Requirements

- Form inputs phải có `<label>` liên kết
- Color contrast ratio > 4.5:1
- Keyboard navigation cho tabs (Arrow keys, Tab)
- ARIA labels cho icon-only buttons
- Focus indicators visible
- Touch target tối thiểu `44px` (`min-h-[44px]`)
